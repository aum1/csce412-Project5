Typescript

Typescript is a programming language way to write JavaScript with additional typechecking in order to remove additional errors at runtime. Catching errors at compile time can mean that there is less unnecessary running of code or wasting of resources.

Some advantages of Typescript include type checking, more detailed error messages, and type inference. Type checking is important in order to make sure that variables are being passed correctly. For example, I was trying to pass an integer when I should have been passing a string into a certain function. However, TypeScript saved me a lot of debugging time because it showed me my error before I even had to run the code. Detailed error messages are also important in order to make your debugging even easier. Type inference also allows for the compiler to fill in type mistakes that the programmer forgets to add.
